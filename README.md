# 🛒 Supermarket System - Codemagic Compatible

مشروع إدارة السوبر ماركت الحديث المتوافق مع **Codemagic CI/CD**

## ✨ المميزات

- 🔥 **Firebase Integration** - Authentication & Firestore
- 📱 **Modern Flutter UI** - Material Design 3
- 📊 **Complete POS System** - Point of Sale
- 📈 **Reporting & Analytics** - Financial Reports
- 🏷️ **Barcode Scanner** - Product Management
- 🖨️ **PDF Export** - Receipts & Reports

## 🚀 البناء على Codemagic

### إعداد المشروع
1. ارفع المشروع إلى GitHub
2. اربط GitHub بـ Codemagic
3. اختر ملف `codemagic.yaml`
4. ابدأ البناء!

### الأوامر البديلة
```bash
# البناء المحلي
flutter pub get
flutter build apk --debug

# اختبار على المحاكي
flutter run
```

## 📱 Build Types

| النوع | الملف | الوصف |
|-------|-------|-------|
| Debug | `app-debug.apk` | للتطوير والاختبار |
| Release | `app-release.apk` | للإنتاج |
